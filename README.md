# Filtro de Números Pares

## Descripción
Aplicación de consola en Java que elimina todos los números pares de una lista utilizando un Iterator.

## Funcionalidades
- Introducir números en una lista
- Eliminar automáticamente los números pares
- Mostrar la lista final solo con números impares

## Tecnologías
- Java
- ArrayList
- Iterator
- Scanner
